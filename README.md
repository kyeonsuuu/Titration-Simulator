# Titration Simulator

가상 중화적정(Titration) 실험을 수행할 수 있는 교육용 시뮬레이터입니다.

본 프로그램은 중화적정 과정과 pH 변화를 시각적으로 탐구할 수 있도록 개발되었습니다.

## Download

### Windows
- Download: `titration-windows.exe`

### macOS
- Download: `titration-macos.zip`

또는 오른쪽의 **Releases** 메뉴에서 최신 버전을 다운로드할 수 있습니다.

## Features

- 강산-강염기 중화적정 시뮬레이션
- 적정 곡선 시각화
- 실시간 pH 변화 확인
- 교육 및 연구 목적 활용 가능
